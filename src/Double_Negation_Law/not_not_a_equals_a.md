### **Double Negation Law**

# **`NOT NOT A equals A`**

---

### Boolean Expression

# **`!!A = A`**

## if A = 0

# **`!!0 = 0`**

## if A = 1

# **`!!1 = 1`**

---

```javascript
!!A === A
```

---

```javascript
let A = true;
console.log(!!A === A); // true

A = false;
console.log(!!A === A); // true
```

---

```javascript
function double_negation(A)
{
    return !!A;
}
```
