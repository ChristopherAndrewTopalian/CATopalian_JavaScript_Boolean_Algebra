# Identity Law  
# **`A OR 0 equals A`**

---

### Boolean Expression  
# **`A + 0 = A`**

## if A = 0  
# **`0 + 0 = 0`**

## if A = 1  
# **`1 + 0 = 1`**

---

```javascript
Number(A) + 0 === Number(A)
```

---

```javascript
let A = true;
console.log(Number(A) + 0 === Number(A)); /* true */

A = false;
console.log(Number(A) + 0 === Number(A)); /* true */
```

---

```javascript
function a_plus_0_equals_a(A)
{
    return Number(A) + 0 === Number(A);
}
```
