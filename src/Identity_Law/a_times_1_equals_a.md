# Identity Law  
# **`A AND 1 equals A`**

---

### Boolean Expression  
# **`A * 1 = A`**

## if A = 0  
# **`0 * 1 = 0`**

## if A = 1  
# **`1 * 1 = 1`**

---

```javascript
Number(A) * 1 === Number(A)
```

---

```javascript
let A = true;
console.log(Number(A) * 1 === Number(A)); // true

A = false;
console.log(Number(A) * 1 === Number(A)); // true
```

---

```javascript
function a_times_1_equals_a(A)
{
    return Number(A) * 1 === Number(A);
}
```
